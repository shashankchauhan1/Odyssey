# Odyssey - Trip Planner

## Project Description
Odyssey is a full-stack Next.js application designed to provide a seamless travel journaling experience. You can Search the Destination and all the required information which helps in your journey will be infront of you within few seconds like weather, local pharases, Modes to reach there, history about destination, Insiders tips, Upcoming Events and much more.

## Project Goals
* **Unified Journaling:** A centralized digital hub for capturing travel memories.
* **Contextual Integration:** Real-time weather data fetching based on trip location.
* **Responsive Performance:** Fast, SEO-friendly rendering using Next.js.

## Specifications
* **Framework:** Next.js 14+ (App Router)
* **Styling:** Tailwind CSS
* **Database:** MongoDB Atlas (via Mongoose/Prisma)
* **Auth:** NextAuth.js / Clerk
* **External APIs:** WeatherStack API

## Design Highlights
* **Architecture:** Three-tier Monolithic Architecture.
* **Database Design:** Document-based storage for flexible journal metadata.
* **Resilience:** Implements basic fault tolerance for external API failures.
